# 🚗 Honda Sales Data Analysis Dashboard (Power BI)

## 📊 Project Overview

This project presents an **interactive Honda Sales Dashboard** created using **Power BI**.
The goal of this project is to analyze Honda vehicle sales data to uncover insights about **sales performance, product trends, and regional distribution**.

The dashboard provides a clear and interactive view of key business metrics that help in **data-driven decision making**.

---

## 📁 Project Files

| File                    | Description                   |
| ----------------------- | ----------------------------- |
| `Image`                 | Image's |
| `Honda_Dashboard.pbix`  | Power BI dashboard file       |
| `Honda All Dataset.csv` | Raw dataset used for analysis |


---

## 🎯 Project Objectives

The main objectives of this project are:

* Analyze **Honda vehicle sales performance**
* Identify **top selling models**
* Understand **regional sales distribution**
* Evaluate **sales trends and patterns**
* Build an **interactive Power BI dashboard**

---

## 📊 Dataset Description

The dataset contains information about Honda vehicle sales including:

* Vehicle Model
* Sales Amount
* Sales Region
* Sales Date
* Dealer Information
* Quantity Sold
* Other related sales attributes

This dataset is used to perform **data cleaning, transformation, and visualization**.

---

## 📈 Dashboard Features

The Power BI dashboard includes several analytical views:

### 1️⃣ Sales Overview

* Total Sales
* Total Units Sold
* Overall Performance Indicators

### 2️⃣ Model Performance

* Top selling Honda models
* Sales comparison between models

### 3️⃣ Regional Analysis

* Sales distribution by region
* Region-wise performance comparison

### 4️⃣ Trend Analysis

* Monthly or yearly sales trends
* Growth patterns over time

### 5️⃣ Interactive Filters

Users can filter dashboard data based on:

* Model
* Region
* Time Period
